# AI项目材料库
本仓库为项目开发素材总仓库，用于配合WorkBuddy实现项目全流程开发，统一存放产品、技术、UI三类开发原材料，可按需调取复用。

## 📁 仓库目录结构
1. **prd-docs（产品需求文件夹）**
存放产品需求文档、需求调研、业务流程图、项目立项说明，是技术开发与UI设计的统一依据。

2. **tech-design（技术方案文件夹）**
前后端技术实现方案、接口文档、代码模板、部署脚本、依赖配置、系统架构设计相关资料。

3. **ui-spec（UI设计文件夹）**
HTML可视化原型页面、UI设计稿、组件设计规范、配色规范、页面交互说明、静态前端素材。

## 🎯 使用说明
1. 新建项目时，可从本仓库复用历史文档、代码模板、设计规范，大幅减少重复开发工作；
2. 配合WorkBuddy绑定GitHub后，可直接读取仓库内文件，完成PRD撰写、技术方案设计、UI规范输出；
3. 每次新增素材请按文件夹分类存放，提交时备注更新内容，方便后续检索调取。
